Install Guide: Running ScriptUI Panels in After Effects

  1. Download the Script Files:

    First, download the ScriptUI panel script file (.jsx or .jsxbin) from the source where you obtained it. Make sure to choose the version compatible with your After Effects software.

  2. Locate After Effects Scripts Folder:

    Open your file explorer (Finder on macOS or File Explorer on Windows) and navigate to the After Effects scripts folder. The location of this folder may vary depending on your operating system and After Effects version. Typically, you can find it at:

    • Windows: C:\Program Files\Adobe\Adobe After Effects [version]\Support Files\Scripts
    • macOS: /Applications/Adobe After Effects [version]/Scripts

  3. Copy Script Files to Scripts Folder:

    Copy the downloaded ScriptUI panel script file into the After Effects scripts folder that you located in the previous step. Paste it directly into this folder.

  4. Restart After Effects:

    If After Effects is currently running, close and restart it to ensure that the newly added script is recognized.

  5. Open ScriptUI Panel:

    After Effects should now be able to detect the newly added ScriptUI panel. To open it, go to the Window menu in After Effects and choose the name of the script panel you installed. It should appear as an option in the list of panels.

  6. Authorize and Run:

    When you open the ScriptUI panel, you may be prompted to authorize the script or provide any necessary permissions. Follow the on-screen instructions to do so. Once authorized, you should be able to run the script and access its features.

  7. Support:

    For additional information, please check our FAQ section. Additional support can be found on the Adobe site. Alternatively, you can contact us here.

Install Guide: Running TVPX or George in TVPaint

  1. Download the Script Files:

    First, download the panel script file (.tvpx or .grg) from the Animeasy. Make sure to choose the version compatible with your TVPaint software.

  2. Locate TVPaint Scripts Folder:

    .grg files can be installed anywhere on your computer, as they are simple ASCII files. However, it's recommended that you install them in the below folders, as any panels that call for a George script will look here. Open your file explorer (Finder on macOS or File Explorer on Windows) and navigate to the TVPaint scripts folder. You can find it at:

    • Windows: C:\Program Files (x86)\TVPaint Developpement\TVPaint Animation [version]\Scripts
    • macOS: /Applications/TVPaint Animation [version]/Scripts
  3. Copy Script Files to Scripts Folder:

    Copy the downloaded ScriptUI panel script file into the TVPaint scripts folder that you located in the previous step. Paste it directly into this folder.

  4. Restart TVPaint:

    If TVPaint is currently running, close and restart it to ensure that the newly added script is recognized.

  5. Go to Panel > Install Custom Panel:

    TVPaint should now be able to detect the newly added panel. To open it, go to the Panel menu in TVPaint and choose the name of the script panel you installed. It should appear as a file in the folder you copied it into.

  6. Authorize and Run:

    When you open the TVPX panel, you will see a splash screen with the copyright information and a set of instructions on how to use the panel. Follow the on-screen instructions to do so. Once authorized, you should be able to run the script and access its features.

  7. Support:

    For additional information, please check our FAQ section. Additional support can be found on the TV Paint Forums. Alternatively, you can contact us here.
